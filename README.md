# 番茄钟

一个功能丰富的番茄工作法计时器，单 HTML 文件，无需依赖，打开即用。

## 功能

- 专注 / 短休 / 长休三种模式，时长可自定义
- 倒计时与正向计时自由切换
- 左侧任务清单，支持主任务与子任务，自动累计专注时长
- 右侧实时时钟与富文本备忘录
- 分层导航设置面板（计时 / 外观 / 通知）
- 6 种主题色调，支持自定义背景图片
- 数据统计：今日概览、任务占比环形图、任务时段分布、周月年折线图
- 专注记录列表，默认折叠
- 低于 3 分钟的专注不记录
- 计时中锁定模式切换
- 补水提醒，可调间隔，支持自定义提醒视频
- 呼吸放松模式：休息时圆环引导呼吸节奏
- 计时运行时圆环波纹扩散效果
- 倒计时数字点击直达计时设置
- 桌面通知 + 提示音
- 休息结束不自动开始专注
- localStorage 持久化
- 键盘快捷键

## 使用

```bash
git clone https://github.com/ShelphenLyon/pomodoro.git
```

浏览器打开 `pomodoro.html`。

## 操作

| 操作 | 快捷键 |
|------|--------|
| 开始/暂停 | 空格 |
| 重置 | R |
| 跳过 | S |
| 切换模式 | 点击顶部标签 |
| 正/倒计时 | 计时器上方开关 |
| 设置 | 右上角齿轮 |
| 统计 | 右上角图表 |
| 任务清单 | 左侧标签 |
| 时钟备忘录 | 右侧标签 |
| 计时设置 | 点击倒计时数字 |

## 技术

原生 HTML / CSS / JavaScript，零依赖，单文件。
